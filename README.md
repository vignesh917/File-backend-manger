File Manager Backend API

A simple backend API for uploading, listing, downloading, and deleting files using **Node.js, Express, Multer, and MySQL**.

Features

- Upload files
- Store file details in MySQL
- View all uploaded files
- Download files
- Delete files

---
 Tech Stack

- Node.js  
- Express.js  
- MySQL  
- Multer  
- dotenv
